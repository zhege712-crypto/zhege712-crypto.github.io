这是你私人使用的页面，没有链接到网站导航里，但网址是公开可访问的，别把这个链接分享给别人。
⚠️ 密钥（Token）具有修改你仓库内容的权限，请不要在公共/他人电脑上勾选"记住密钥"，也不要把密钥截图发给任何人（包括我）。

① 仓库与密钥设置

还没有密钥？去 GitHub → 头像 → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token。
Repository access 选 "Only select repositories" 只勾你的网站仓库； Permissions 里把 Contents 设为 Read and write，其他保持默认即可。

图片会被上传到仓库的 photos/poems/ 文件夹下。

③ 管理已发布内容

填好上面①的仓库信息和密钥后，这里会自动列出所有已发布的诗，点删除即可移除（配图也会一并删除）。
